# sanoj.dev

[![Portfolio uptime check](https://github.com/d-sanoj/sanoj.dev/actions/workflows/uptime.yml/badge.svg)](https://github.com/d-sanoj/sanoj.dev/actions/workflows/uptime.yml)

Personal portfolio site for Sanoj Doddapaneni.

## Overview

This is a lightweight static site built with plain HTML, CSS, and JavaScript. It includes portfolio content, visual effects, and local image assets.

## Files

- `index.html` - main page
- `styles.css` - site styling
- `script.js` - interactions and UI behavior
- `fluid-cursor.js` - cursor/canvas effect
- `assets/` - images, icons, and certification badges

## Local Preview

Open `index.html` in a browser.

## Uptime Monitoring

The status badge above shows the latest hourly uptime check for `https://sanoj.dev/`.
Click the badge to open the GitHub Actions run history.

To get notified when the uptime check fails, enable GitHub Actions notifications:

1. Open the repository on GitHub.
2. Go to **Settings** > **Notifications**.
3. Under **Actions**, enable failed workflow notifications.
