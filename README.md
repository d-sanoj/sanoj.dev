# sanoj.dev

Personal portfolio site for Sanoj Doddapaneni.

## Overview

This is a lightweight static site built with plain HTML, CSS, and JavaScript. It includes portfolio content, visual effects, and local image assets.

## Files

- `index.html` - main page
- `styles.css` - site styling
- `script.js` - interactions and UI behavior
- `fluid-cursor.js` - cursor/canvas effect
- `assets/` - images, icons, and certification badges

## Local Preview

Open `index.html` in a browser.
